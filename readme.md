# 変なブロック崩し

ボールがブロックやバーに当たると、直感に反してランダムに跳ね返るブロック崩しです。
HSP 製。

2011 年の高専祭で展示しました。

## 操作方法 (キーボードのみ)
　[←]/[→]: バーの左右移動
　[SPACE]: ボールの発射
　[ESC]: 強制的にゲームオーバー

## 仕様
- ステージ: 全 6 面
- ボール
	- 右上、右下、左上、左下の 4 方向に移動
	- ブロックやバーに衝突するとランダムに移動する方向が決まる
	- ボールが画面下端まで移動した時のペナルティ
		- スコア: 5 % 減
		- 残り時間: 10 % 減
- ブロック
	- 種類
		- 黄金ブロック (黄色)
		- 通常ブロック (黄色以外)
	- どちらも 1 度ボールを当てると消える
- スコア
	- 黄金ブロック (黄色) → 残り時間の 10 % が加算される
　　- 通常ブロック (黄色以外) → 5 点加算
	- 各ステージのハイスコアが記録される
- 制限時間
	- ブロック 1 つにつき約 5 秒
- 終了条件
	- ブロックをすべて破壊するとゲームクリア
	- 制限時間が 0 になるか、3 ミス以内にゲームクリアできないとゲームオーバー

## 既知のバグ
- [Alt]+[F4]すると落ちる
